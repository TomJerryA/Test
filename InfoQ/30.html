Node.js Tools for Visual Studio Beta 1发布

Microsoft最近发布了一个beta版本的Visual Studio Node.js工具，该工具也被称为NTVS。值得注意的是这个版本支持免费版本的Visual Studio，正式名称是Visual Studio Express for Web。

为V8 JavaScript引擎集成调试器支持的工作仍在继续。Somasegar写道“NTVS中的调试支持用户期望Visual Studio具有的所有核心调试功能：断点、调用栈、本地变量、查看、条件断点、过滤器、when-hit操作等”。本地及Azure上的Windows或Linux服务器都支持调试。

这个版本新增了编辑并继续支持。与其他语言及运行时一样，这意味着修改代码的时候不再需要重启服务器。

该版本的另一个新特性是它支持最近发布的TypeScript。

Somasegar继续写道：

该版本中的另一个新特性是它支持Azure 辅助（Worker）角色。开发者现在能够使用集成的工具部署到Azure Web网站或者Azure辅助角色。通过使用辅助角色，你可以构建与前端Web端点松耦合的托管后端服务。

Node Tools for Visual Studio 是基于Apache许可发布的一个开源项目。该项目由Microsoft、RedGate和Clickberry三家公司联合开发。
