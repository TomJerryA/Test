TechEd 2013：通过Cloud OS、Visual Studio等构建微软生态社区

2013年12月5日，TechEd 2013大会在北京召开，这次大会围绕“创新、开放、社区”为主题， 来自云微软和相关合作伙伴、社区的技术专家在会上和大家分享了微软的最新技术进展，今年是TechEd进入中国20年，微软在会上正式宣布启动第二届微软云创益大赛，开发者可以通过大赛官网报名参赛。

大会相关数据

  • 微软企业级业务目前占微软全球业务的55%
  • 55%的《财富》500强企业使用Windows Azure
  • 企业社交软件Yammer现拥有超过800万的注册用户
  • Windows Azure上约有8.5亿个伟健
  • Visual Studio 2012累计下载突破600万次
  • 74家合作伙伴提供123款Visual Studio 2013的延伸产品

Cloud OS

作为同时提供跨越私有云、公有云和混合云服务的企业，微软2012年提出了“三云合一”统一管理的Cloud OS云操作系统理念，并于今年11月15日面向国内用户正式开始发布，Clouds OS以Windows Azure、Windows Sever和System Center为核心，通过Hyper-V虚拟化技术，实现大规模虚拟化、高性能存储、软件定义网络（SDN）和混合业务连续性方案。

在存储方面，Windows Server 2012 R2的Storage Spaces使用SSD固态硬盘加HDD硬盘提供基于策略的分层功能，降低存储的成本；另外，Hyper-V提供了诸如点对点VPN网关等内置功能，桥接物理网络和虚拟网络，通过System Center 2012 R2进行负载迁移。

就现在看来，微软全球的Azure用户以企业级用户为主，其中，Azure自6月6日在国内开发公测以来，已经拥有2000余家用户，其中70多家是付费用户，目前，微软还在考察游戏等市场的潜力。

为了促进Clouds OS的普及，微软启动了“以全云，筑创想”为主题的第二届微软云创意大赛，并于今天正式接受报名。

Visual Studio

会上，微软全球开发平台事业部全球资深副总裁潘正磊女士分享了微软的Visual Studio的一些数据和团队经验，潘提到自一年前Visual Studio 2012发布以来，它已经成为微软有史以来普及速度最快的Visual Studio版本，目前下载量已经突破了600万次，在VS 2012发布后的14个月中，微软还发布了Visual Studio 2013、Visual Studio Online已经Visual 2012的4个版本更新。

Visual Studio 2013加入了诸如Azure网站实时调试、能耗报告、用户界面相应诊断等功能，同时，Visual Studio 2013以及Team Foundation Server 2013还提供了敏捷开发的支持，通过结合Windows Server 和Systems Center，推动DevOps的发展。

潘说到市场的变化太快，为了迎合新的机遇和挑战，Visual Studio开发团队通过敏捷的方式加速产品的开发进程，目前Visual Studio开发团队以3个星期作为一个Sprint、6个月计划一个Season、并使用18个月实现一个Vison。他们将整个团队分拆成一个个实体的功能团队，每个团队有12到18个人，自主掌握开发进度和功能需求。

开发者支持

开发者生态系统是这次大会的主题之一，目前微软和中国移动、联通、支付宝等企业达成了合作协议，Windows 8以及Windows Phone平台软件可以通过相关的SDK实现移动支付。微软还和Unity3D、cocos2D-x、Xamarin建立了合作伙伴关系，为开发者提供游戏开发引擎工具和技术支持。
