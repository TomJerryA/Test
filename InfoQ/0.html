微软携手RedGate和Clickberry提供Node.js Tools for Visual Studio

NTVS是Node.js Tools for Visual Studio的缩写，它为Node.js开发人员提供了一个带有智能提示、包管理、性能分析及调试功能的全功能IDE。

NTVS既支持本地调试，也支持远程调试。在远程调试的情况下，它还支持MacOS和Linux。调试器的大部分工作是由Clickberry的Dmitry Tretyakov完成的。

包管理通过NPM，也就是Node Packaged Modules，进行处理。Red Gate的Bart Read为该功能提供了用户界面。

出现如此多的第三方是因为多个支持Node.js的项目开始于大致相同的时间。而且，它们在很大程度上基于Python Tools for Visual Studio（PTVS）。因此，当微软的PTVS团队意识到发生了什么的时候，他们决定征集社区贡献。一个名为dinov的Reddit用户说：

我在微软从事NTVS方面的工作，它主要由微软构建。但是，在发布之前，我们也与其他组织或个人进行了合作。

在从事这项工作的过程中，我们看到了几个为VS提供Node.js支持的不同尝试，包括RedGate的Visual Node和由一些个人开发的另一个“NTVS”。在发布之前，我们联系了他们，向它们开放了代码，并很高兴地接受了他们的贡献。最值得注意的贡献是NPM支持，这是来自RedGate的一个非常重要的功能。

由于它使用了内建的JavaScript编辑器，NTVS需要Visual Studio专业版或者更高版本。
