Visual Studio信息汇总

Visual Studio Update 2的二月份CTP

Microsoft继续为即将到来的Visual Studio 2012 Update 2推出更新。最新版的Update 2 CTP 3已经提供下载。如同以往的版本，新版本还不适合部署到生产环境中。不过来自Microsoft的Brian Harry指出，下一个的CTP就将能够部署到生产环境了。
最新的CTP中持续添加了新的特性。首先的是Visual Studio Premium版本中加入了对“Fakes”的支持（译注：Microsoft Fakes是一套可以在Unit Test中自动生成Stub或Mock的类库），以前只有Ultimate版本支持该特性。Microsoft Test Manager现在支持对整个测试计划的克隆，这使得重用已有计划更加方便。而使用HTML和JavaScript的开发者也将从新增的“UI响应工具”（又名Visual Profiler）中受益。
之前的CTP中引入的基于web的测试用例管理工具，现在不但支持对某些时间过长的（几天或几周）测试进行暂停和恢复的操作，还能在机器重启恢复测试。基于web的测试用例现在也能够添加附件了。

DevLabs移至Visual Studio库

长久以来，Visual Studio Gallery都用于发布Visual Studio的各种插件和扩展。本周Microsoft宣布，它已经把DevLabs门户从MSDN移至Gallery。现在，DevLabs不再是个独立的部分，可以通过Gallery找到它。
就像Microsoft的S.Somasegar所说，有些项目的首次亮相就是出现在DevLabs上，包括：

  • Code Contract
  • CHESS，一个并发测试的工具
  • Pex and Moles，另一个测试工具
  • Axum，一门实验性质的.NET 语言，用于创建并行应用程序（该语言的二进制包现在仍然可以下载）
