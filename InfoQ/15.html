新的SQL Server、SharePoint和Office开发工具

对于想要开发SQL Server 2012商务智能项目的开发者而言，现在能够使用最新发布的Microsoft SQL Server Data Tools—— 商务智能 (SSDT-BI) 软件进行开发了。最新发布的工具适用于SQL Server 2005、 SQL Server 2008、 SQL Server 2008 R2、 SQL Server 2012和SQL Azure。

开发者依然会喜欢我们为数据库开发所提供的这些类似于VS的工具，特别是我们提供了与C#和VB中已有功能相对应的代码导航、智能感知和语言支持，为TSQL编辑器提供了平台特定的验证、调试和声明式编辑支持，还为数据库项目和在线数据库实例提供了可视化表编辑器。

最新版本的工具支持完全往返的项目，因此用户在升级到2012之后依然能够操作基于2010的文件。使用VS2012简化版的用户也能够通过另外的安装包安装BI。另一个期望的特性是，该版本的工具集将和VS2012 GUI外形（shell）保持一致。

Office开发者工具

Office团队并没有被超越，因为他们也发布了VS2012的开发者工具 （注意这是下载安装文件的链接）。开发者能够通过这些工具创建Office和SharePoint应用。最新版本的工具支持.NET Framework 4.5，能够为Office 2013构建应用，同时还能远程调试SharePoint网站。那些之前通过人工方式创建引导程序包从而提供.NET 4.5功能的开发者，在这个版本中不再需要这样做了。注意，和上面的BI工具不同的是，Office开发者工具需要VS2012专业版或者更高版本。
