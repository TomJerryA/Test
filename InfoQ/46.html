Typemock Isolator v7.4.3增加了FastFix，改进了对SmartRunner和Visual Studio 2013的支持

用于.NET的单元测试工具Typemock Isolator已经发布了v7.4.3版本，除了新的测试预览窗口，该版本还增加了对FastFix的支持，并增强了对SmartRunner、Visual Studio 2013和Team Foundation Server (TFS) 2012的支持 。在FastFix的帮助下，相关测试不仅与调试绑定了起来 ，还可以确保用户看到所有测试状态，所以开发人员就不必在文件和视图之间切换了。

最新版本改进了SmartRunner的机制， 使得缓慢的集成测试可以在空闲时间执行，而较快的单元测试则直接执行。该版本还包括一个新的测试预览窗口，可以在同一屏内显示测试代码和被测试代码，此外还有更新的菜单、高亮显示特性，并增强了可用性。

官方消息称，通过节省宝贵的开发时间，TypeMock中的自动化单元测试机制消除了产品代码中的回归缺陷，还改进了测试驱动场景下的代码设计。
