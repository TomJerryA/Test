Red Gate 发布ASP.NET MVC学习门户网站——Simple Web Dev

Red Gate 软件公司最近发布了Simply Web Dev,这是一个新的教学辅导门户网站，能让开发者学习ASP.NET MVC。该网站主要提供了与ASP.NET MVC相关的外部资源链接，如 区域（Areas）、路由（Routing)、 控制器（Controllers）和操作（Actions）、 控制器扩展（Controller Extensibility）、 过滤器（Filters）、视图（Views）、 模型模板（Model Template）、绑定（Binding）和验证（Validation）。

InfoQ采访了 Red Gate软件公司的产品营销专家Chris Massey，以了解更多与Simple Web Dev门户网站相关的详细信息。

InfoQ：您能与我们分享这个门户网站导学指南的意义吗？

导学指南汇总了目前网上的一些最佳的技术文章，这些文章由经验丰富的ASP.NET开发人员策划汇总，以便为那些学习ASP.NET MVC的新手程序员提供最丰富翔实的内容。

简而言之，指南的目的是帮助学习ASP.NET MVC的开发人员从经验丰富和更有洞察力的Web开发人员那里获益，让他们仅需要阅读那些学习该技术所需要的知识。

InfoQ：您能告诉我们创建Simple Web Dev门户网站的必要性吗？

如果你想学习ASP.NET MVC，必须浏览网上现有的大量资源才能找到自己所需要的内容。这些内容有很多是重复的，而且质量也参差不齐，如果你不是一个经验丰富的Web开发人员就很难找到真正有用的内容。

我们认为该门户网站对于新的Web开发人员和那些新学习ASP.NET的开发人员来说是很有用的，能够帮助他们很快地找到一些最好的学习资源，帮助他们了解技术并使其相信自己阅读的内容的质量。

InfoQ：Red Gate创建Simple Web Dev门户网站的灵感来自于哪里？

我们最近一直在寻找支持Web开发社区的方法。当然，通过我们的作者网络，我们可以委托他们发表更多的文章和博客帖子去满足web开发人员的需要，但是当我们开始寻找已经有的内容时，我们萌生了另外的想法。

我们很快就意识到，已经有大量已发布的资源，除了再添加新的资源之外，我们还能够通过另外一种方式帮助Web开发人员，那就是提供一种途径（人工专家）使其能够找到最好的内容和人们智慧的结晶。不要误会我的意思，Google是一个优秀的寻找信息的工具，但是此时有一个人工专家告诉你应该阅读哪些内容依然是一个更好的方式。

InfoQ：你们有没有计划为Visual Studio 2012、ASP.NET和LINQ建立类似的门户网站？

当然。我们计划扩大导学指南的覆盖范围，并且没有主题的限制。然而，我们的首要任务是帮助Web开发人员更好地了解他们正在使用的技术，所以在接下来的几月中我们可能不会开始策划针对开发工具的指南，除非有人给我们发送电子邮件并说服我们。

INFO:你们打算多久更新一次Simply Web Dev？

我们的目标是每周对网站进行更新以增加更多的内容。首先，我们正在努力完成对ASP.NET MVC的覆盖，并使用它作为初始原型帮助我们更好地了解开发人员对内容的需求，以及如何才能改善我们所策划资源的功能和用户体验。

当我们对如何支持社区有更多的信心之后，我们将开始加速并发布更多的指南。

InfoQ：您认为Simple Web Dev门户网站对学生有帮助么？

当然有。如果你是一个刚开始学习Web开发的学生，或只是刚开始学习ASP.NET技术，有一个领域专家能够向你讲解深入理解该技术需要阅读哪些内容，以及为什么要阅读这些内容，这种收益无疑是巨大的。
我们想帮助那些正在学习web开发的人，让他们能够尽快地、平缓地理解他们正在使用的技术，不需要很多搜索，并且我们认为这对学生特别有帮助。
