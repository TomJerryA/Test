WSO2 Developer Studio 3.0&3.1改进了ESB工具

WSO2 Developer Studio 3.0、3.1版本改进了ESB和注册资源编辑器的工具,并拥有更高的性能，它使用Eclipse Juno SR2,并支持JAX-RS REST代码生成。

WSO2近期发布了Developer Studio IDE(WSO2 DS)的两个新版本：3.0和3.1。他们最初的计划是发布一个大版本，但一周后,WSO2又设法发布了一份小版本，其中根据客户要求做了多项改进。

WSO2 DS延续了从2.0版本确立的趋势：简化Carbon和Stratos应用开发。WSO2 DS IDE 3.0和3.1中的主要提升有：

  • WSO2 ESB图形化流结构工具——一种图形化工具，有助于展现连接到ESB的不同组件之间的消息流路径
  • WSO2 ESB REST API支持——在Carbon Archive（CAR）中，自动打包ESB REST服务
  • WSO2 ESB任务支持——用于安装同样在CAR中打包的重复性任务
  • 注册资源编辑器现在可以用于在不同环境间迁移媒体资源
  • 更好的整体性能
  • 支持Eclipse Juno SR2
  • 支持使用WADL生成基于JAX-RS的REST代码

WSO2 Developer Studio基于Eclipse的IDE运行，它提供用于创建SOA或复合应用的集成工具，以及RESTful服务——可以本地化部署到Carbon中间件平台，也可以部署到基于Stratos的私有或混合云，还可以部署到由WSO2运营的名为StratosLive的Java PaaS上。
